API Documentation


1) Account Services

1) Get the basic information for an account.
@GET : /account
 
2) Get the lists that you have created and marked as a favorite.
@GET : /account/lists
 
3) Get the list of rated movies (and associated rating) for an account.
@GET : /account/rated/movies
 
4) Get the list of favorite movies for an account.
@GET : /account/favorite/movies
 
5) Get the list of movies on an accounts watchlist.
@GET : /account/watchlist/movies
 
6) Rate a movie.
@POST : /account/rate/movie?movieid=&rating=
 
7) Add a movie to an account's favorites list.
@POST : /account/add/favorite/movie?movieid=
 
8) Remove a movie from an account's favorites list.
@DELETE : /account/add/favorite/movie?movieid=
 
9) Add a movie to an account's watch list.
@POST : /account/add/watchlist/movie?movieid=
 
10) Remove a movie from an account's watch list.
@DELETE : /account/add/watchlist/movie?movieid=
 
 

2) Genre Services

1) Browse all genres.
@GET : /genre/getgenrelist
 
2) Browse all movies in a particular genre.
@GET : /genre/getgenremovies/{genreid}
 
 

3) List Services

1) Create a new list.
@POST : /list/create?name=&description=
 
2) Delete an existing list.
@DELETE : /list/delete?listid=
 
3) Add new movies to a list.
@POST : /list/add?listid=&movieid=
 
4) Delete movies from a list.
@DELETE : /list/remove?listid=&movieid=
 
5) Check to see if a movie ID is already added to a list.
@GET : /list/status?listid=&movieid=
 
 

4) Movie Services

1) Get a movie by id.
@GET : /movies/getmovie/{movieid}
 
2) Get alternative titles for a movie.
@GET : /movies/getmoviealternativetitles/{movieid}
 
3) Get credits for a movie.
@GET : /movies/getmoviecredits/{movieid}
 
4) Get Images for a movie.
@GET : /movies/getmovieimages/{movieid}
 
5) Get Keywords for a movie.
@GET : /movies/getmoviekeywords/{movieid}
 
6) Get Release Info for a movie.
@GET : /movies/getmoviereleaseinfo/{movieid}
 
7) Get Videos for a movie.
@GET : /movies/getmovievideos/{movieid}
 
8) Get Translations for a movie.
@GET : /movies/getmovietranslations/{movieid}
 
9) Get a list of similar movies for a movie.
@GET : /movies/getmoviessimilar/{movieid}
 
10) Get a list of upcoming movies.
@GET : /movies/getmoviesupcoming
 
11) Get a list of now-playing movies.
@GET : /movies/getmoviesnowplaying
 
12) Get a list of popular movies.
@GET : /movies/getmoviespopular
 
13) Get a list of top-rated movies.
@GET : /movies/getmoviestoprated
 
14) Get a list of forthcoming movies.
@GET : /movies/getmoviesforthcoming
 
 

5) People Services

1) Get the general person information for a specific id.
@GET : /people/person?personid=
 
2) Get the credits for a specific person id.
@GET : /people/credits?personid=
 
3) Get the images for a specific person id.
@GET : /people/images?personid=
 
4) Get the list of popular people on The Movie Database.
@GET : /people/popular
 
 

6) Search Services

1) Search for movies by title.
@GET : /search/movie/{title}
 
2) Search for people by name.
@GET : /search/person/{name}
 
3) Search for keywords by name.
@GET : /search/keyword/{name}
 
4) Search for lists by name and description.
@GET : /search/list/{name}
 
 

7) Twitter Services

1) To get request token for twitter.
@GET : /twitter/request?user=
 
2) To get access token for twitter.
@GET : /twitter/success?oauth_token=&oauth_verifier=
 
3) To post in twitter on behalf of a particular user.
@GET : /twitter/status?user=
 
4) To post in twitter on behalf of all users.
@GET : /twitter/post
 
 

8) User Services

1) To register a new user.
@POST : /user/register?username=&password=&firstname=&lastname=&gender=
 
2) To login to the application.
@GET : /user/login?username=&password=
 
3) To get TMDb request token.
@GET : /user/getrequesttoken
 
4) To get TMDb access token.
@GET : /user/getaccesstoken
 
5) To save TMDb access token.
@POST : /user/saveaccesstoken?sessionid=
 
6) To logout of the application.
@GET : /user/logout
 
7) To delete an existing user.
@DELETE : /user/deregister
